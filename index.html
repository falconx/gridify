Gridify | jquery.gridify.js

Need a collection of elements to appear in a grid-like fashion? The Gridify jQuery plugin makes this simple...

Gridify depends on the smartresize jQuery plugin for smoother window resize performance and should be loaded before the Gridify script.

How to use

Load the jQuery, smartresize and gridify (Keep this order): <script src="http://code.jquery.com/jquery.min.js"></script>
<script src="js/jquery.smartresize.js"></script>
<script src="js/jquery.gridify.js"></script>

Typical markup scenario, but not limited to:
<ul id="items">
  <li>First</li>
  <li>Second</li>
  <li>Third</li>
  <li>Forth</li>
  <li>Fifth</li>
  <li>Sixth</li>
</ul>

You do not need your collection to be a unordered list of course, you may be limited in the markup you have to work with or have other reasons for not choosing a list.
The following example shows how paragraphed content can be styled as a grid, simply by providing a class on the container:
<div class="grid">
  <p>First</p>
  <p>Second</p>
  <p>Third</p>
  <p>Forth</p>
</div>

The JS

$('#items').gridify();

- or -

$('.grid').gridify();

The above could be called on $(document).ready() however,
if your content contains images you will want to use $(window).load() to allow the script to perform calculations on the loaded image dimentions.

Optional Parameters

  • items (how many items per row, defaults 4)
  • children (direct child element identifier, defaults 'li')
$('#items').gridify({ items_per_row: 3, children: '.item' });

Finally...

Apply styling as you would normally.
